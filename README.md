This repository contains the source code for the official website of BluePrints 2026, an intercollegiate technical event focused on innovation, collaboration, and problem-solving.

The website serves as the primary digital platform for event information, announcements, and participant engagement.

**Purpose**

1. Present Official Event Details And Updates

2. Enable Participant Registration And Communication

3. Provide A Centralized And Reliable Source Of Information

**Features**

1. Responsive And Modern User Interface

2. Structured Content For Event Details And Schedules

3. Optimized Performance And Accessibility

4. Scalable Design For Future Updates

**Tech Stack**

1. TypeScript

2. JavaScript

3. CSS
